# yilai
Yeelight Lights Front/Backend Server

Simple Web App to control Yeelight WiFi LED bulbs

## Install

Clone repository
```
git clone https://github.com/pulento/yilai.git
cd yilai
```
Clone yeelight-wifi fork
```
git submodule init
git submodule update
```

Instal server modules
```
npm install
```

And client modules..
```
cd client
npm install
```
Run :)
```
cd ..
npm run dev
```
